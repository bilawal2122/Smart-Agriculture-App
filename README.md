# 🌱 Smart Agriculture App

A modern Flutter application designed to help farmers make informed agricultural decisions using weather forecasting, crop management information, market prices, and AI-powered plant disease detection.

---

## 📖 Overview

Smart Agriculture App is an intelligent farming assistant that provides essential agricultural information in one place. The application empowers farmers with real-time weather updates, detailed crop guidance, fertilizer and irrigation recommendations, current market prices, and plant disease analysis using Machine Learning.

The goal of this project is to improve farming productivity, reduce crop losses, and support data-driven agricultural practices.

# 🚀 Getting Started

## Prerequisites

Make sure you have installed:

- Flutter SDK
- Dart SDK

- Android Emulator or Physical Device

---

## Installation

### Clone the repository

```bash
git clone https://github.com/your-username/smart_agriculture_app.git
```

### Navigate to the project

```bash
cd smart_agriculture_app
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```

# 📷 Machine Learning Disease Detection

The application uses a trained Machine Learning model to identify plant diseases from crop images.

Workflow:

1. Capture or upload a plant image.
2. Image is processed by the ML model.
3. Disease is predicted.
4. The application displays:
   - Disease name
   - Confidence score *(if available)*
   - Recommended treatment and precautions
